Why WhisperKey

🔒

Completely Private

Speech recognition runs entirely on your Mac using Apple Neural Engine. No audio is ever sent to any server.

⚡

Instant Transcription

Powered by WhisperKit — OpenAI's Whisper model optimized for Apple Silicon via CoreML. The whisper-tiny.en model runs on Apple Neural Engine for sub-second English transcription.

🎯

Works Everywhere

Text appears wherever your cursor is — Slack, email, Notes, browser, IDE, any text field in any app.

🎙

Any Microphone

Built-in mic, AirPods, USB headset, external condenser — pick your input device from the menu bar.

⌨️

🪶

Lightweight

2.2 MB download. Lives in the menu bar — no dock icon, no windows cluttering your workspace.

How It Works

1

Install & Grant Permissions

Open the app. Allow Microphone and Accessibility access when prompted. WhisperKey appears in your menu bar.

2

Click into Any Text Field

Position your cursor where you want the text — an email draft, a Slack message, a code comment, anywhere.

3

Hold Fn and Speak

The menu bar icon turns red while recording. Release Fn to stop. Or double-tap Fn to toggle hands-free mode.

4

Text Appears Instantly

Your speech is transcribed on-device and pasted into the active text field. Previous clipboard is restored automatically.

Under the Hood

Model

OpenAI Whisper

whisper-tiny.en variant — 39M parameters, English-optimized

Runtime

WhisperKit + CoreML

By Argmax — Whisper ported to Apple Neural Engine

Processing

100% On-Device

Runs on Apple Neural Engine & GPU. Zero network calls.

Latency

< 1 second

Real-time factor ~0.1x on Apple Silicon M-series chips

Audio Format

16 kHz Mono PCM

Optimal for speech recognition. Any mic input auto-converted.

Text Injection

Accessibility API
